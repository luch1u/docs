## Bienvenido

Esta API permite interactuar con el SII (Servicio de Impuestos Internos) de Chile de forma automatizada. Utiliza Playwright para automatizar las interacciones con el portal web del SII.

## Características

- ✅ Verificación automática del estado de login en SII
- ✅ Emisión automática de boletas de honorarios
- ✅ Validación de RUT chileno con dígito verificador
- ✅ Retorno de PDF de boletas en formato base64
- ✅ Manejo de errores robusto

## Endpoints disponibles

### Health Check
- `GET /` o `GET /health` - Verifica que la API esté funcionando

### SII
- `GET /sii/health` - Verifica el estado de las credenciales del SII
- `POST /sii/boleta` - Emite una boleta de honorarios automáticamente

## Autenticación

Las credenciales del SII se almacenan de forma segura en AWS Secrets Manager y se recuperan automáticamente en cada request. No es necesario enviar credenciales en las peticiones.

## Base URL

La API está desplegada en AWS API Gateway. La URL base depende del stage de despliegue:

- **Desarrollo**: `https://your-api-gateway-url.execute-api.us-east-1.amazonaws.com/dev`
- **Local**: `http://localhost:3000` (usando serverless-offline)

## CORS

Todos los endpoints tienen CORS habilitado y aceptan peticiones desde cualquier origen.
