## Bienvenido

Esta API permite interactuar con el SII (Servicio de Impuestos Internos) de Chile de forma automatizada. Utiliza Playwright para automatizar las interacciones con el portal web del SII.

## Características

- ✅ Verificación automática del estado de login en SII
- ✅ Emisión automática de boletas de honorarios
- ✅ Validación de RUT chileno con dígito verificador
- ✅ Retorno de PDF de boletas en formato base64
- ✅ Manejo de errores robusto

## Endpoints disponibles

### Health Check
- `GET /` o `GET /health` - Verifica que la API esté funcionando

### SII
- `GET /sii/health` - Verifica el estado de las credenciales del SII
- `POST /sii/boleta` - Emite una boleta de honorarios automáticamente

## Autenticación

Para usar esta API, necesitas obtener un token de autenticación. El proceso es el siguiente:

1. Visita [https://kontaiapi.com](https://kontaiapi.com)
2. Inicia sesión con tus credenciales
3. Obtén tu token de API
4. Usa el token en todas tus peticiones incluyéndolo en el header `Authorization`:

```bash
Authorization: Bearer TU_TOKEN_AQUI
```

<Warning>
Sin un token válido, no podrás acceder a los endpoints de la API. Asegúrate de incluir el token en todas tus peticiones.
</Warning>

## Base URL

La API está disponible en el siguiente endpoint:

**Ambiente de desarrollo/pruebas:**
```
https://ql3xbjhoe0.execute-api.us-east-1.amazonaws.com/dev
```

<Tip>
Todos los endpoints deben ser llamados usando esta URL base como prefijo. Por ejemplo: `https://ql3xbjhoe0.execute-api.us-east-1.amazonaws.com/dev/health`
</Tip>

## CORS

Todos los endpoints tienen CORS habilitado y aceptan peticiones desde cualquier origen.
