## Bienvenido

Esta API te permite interactuar con el SII (Servicio de Impuestos Internos) de Chile de forma automatizada. Utiliza Playwright para automatizar las interacciones con el portal web del SII, permitiéndote verificar credenciales y emitir boletas de honorarios sin intervención manual.

## Endpoints disponibles

### Health Check
- `GET /health` - Verifica que la API esté funcionando correctamente

### SII
- `GET /sii/health` - Verifica el estado de las credenciales del SII
- `POST /sii/boleta` - Emite una boleta de honorarios automáticamente

## Autenticación

Para usar esta API, necesitas obtener un token de autenticación. El proceso es simple:

1. Visita [https://kontaiapi.com](https://kontaiapi.com)
2. Inicia sesión con tus credenciales
3. Obtén tu token de API
4. Usa el token en todas tus peticiones

<Warning>
Sin un token válido, no podrás acceder a los endpoints de la API.
</Warning>
